## Why PopUnity Pax? üî•
In 2026, online hate is everywhere ‚Äì echo chambers divide us, polarization is up. But what if your feed actually healed instead of hurt?  

PopUnity Pax is the peace-first app:  
- "Pop" quick positive stories & ideas  
- Get matched with peeps from different views for real talks  
- Join live Pax Rooms with gentle AI prompts to keep it chill & empathetic

## Core Features (MVP Vision)  
1. **Unity Pops** ‚Üí Short media posts (text, pics, vids) on peace themes ‚Äì forgiveness wins, cultural shares, solution ideas.  
2. **Pax Rooms** ‚Üí Live audio/video circles with real-time empathy coaching (AI flags heat, suggests "Share how this feels?").  
3. **Unity Matches** ‚Üí Smart pairing across cultures/politics ‚Äì start with icebreakers for safe convos.  
4. **Global Pax Board** ‚Üí Interactive map of peace actions ‚Äì pin events, reconciliations, collabs.  

**Future vibes**: Multi-lang translation, offline mode, Pop Challenges (gamified quests), NGO event tools.
